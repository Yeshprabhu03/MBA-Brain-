# 📖 How to Use This Vault

This vault is designed as a **second brain for MBA-level business knowledge** — interconnected, searchable, and growing. Here's how to get the most out of it.

---

## 🧭 Navigation Modes

### 1. Structured Learning (Top-Down)
Start from a **subject MOC** and work through it like a course syllabus:
- [[📊 Finance MOC]] → Time Value of Money → DCF → WACC → ...
- [[🎯 Strategy MOC]] → Competitive Advantage → Porter's Five Forces → ...

### 2. Exploratory (Graph-Based)
Open **Obsidian Graph View** (`Ctrl/Cmd + G`) and explore connections visually. Clusters reveal how concepts relate.

### 3. Reference Mode
Jump directly to a concept, framework, or case:
- Use `Ctrl/Cmd + O` to open Quick Switcher
- Use `Ctrl/Cmd + Shift + F` to full-text search across all notes

### 4. Framework Lookup
Go to [[🔧 Frameworks MOC]] to find the right tool for any business problem.

## 🔗 Wikilink Conventions

- `[[Note Name]]` — Link to another note
- `[[Note Name|Display Text]]` — Link with different display text
- `[[Note Name#Section]]` — Link to a specific heading
- Every concept note links back to its parent MOC

---

## 📐 Adding New Notes

Use the templates in `/Templates/`:
1. Open command palette (`Ctrl/Cmd + P`)
2. Type "Templates: Insert template"
3. Choose the appropriate template
4. Fill in all fields
