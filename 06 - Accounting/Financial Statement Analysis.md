# Financial Statement Analysis

The process of evaluating a company's past, present, and future performance using its financial statements.

- **Horizontal Analysis**: Comparing data across time periods.
- **Vertical Analysis**: Expressing line items as a % of a base (e.g., Net Sales).
- **Ratio Analysis**: Liquidity (Current Ratio), Solvency (Debt/Equity), Profitability (ROE, ROA).
