## 🔧 Key Formulas

| Formula | Expression |
|---------|-----------|
| PV | `CF / (1+r)^n` |
| WACC | `(E/V)×Re + (D/V)×Rd×(1−T)` |
| CAPM | `Re = Rf + β(Rm − Rf)` |
| EV | `Equity + Debt − Cash` |
| Terminal Value (Gordon) | `FCF × (1+g) / (WACC − g)` |
